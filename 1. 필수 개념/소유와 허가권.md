파일 소유권(Ownership): 파일을 소유한 사용자와 그룹을 의미한다.

## 파일 유형
- 디렉터리는 d
- 일반 파일이면 -

## 파일 허가권
r(read), w(write), x(execute)
- 첫째 세자리 : 소유자(User)의 파일 접근 권한
- 둘째 세자리 : 그룹의 파일 접근 권한
- 셋째 세자리 : 나머지(Other)의 파일 접근 권한

&nbsp;  

8진수 숫자로 표기 (r:4, w:2)
rw -> (4+2) -> 6

&nbsp;  


### chmod

보안상 위험하여 루트 사용자만 실행 가능한 명령어.

### chown, chgrp : 파일의 소유권 변경
